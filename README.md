# Weather Application

Real-time weather app built with React and OpenWeatherMap API.

## Features
- City search
- Current weather display
- Temperature, humidity, wind speed
- Weather icons
- Loading and error states
- Keyboard support
- Responsive design

## Technologies
- React (Hooks: useState, useEffect)
- OpenWeatherMap API
- Fetch API with async/await
- CSS3
- Responsive design

## Live Demo
[View Live](https://prathiba-weather-app.netlify.app/)

## Installation
```bash
npm install
npm start
```
